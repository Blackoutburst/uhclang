# Standard Library

Welcome to the **UnholyC** standard library reference.

The standard library is organized into theses sections:

- [**IO**  — File, logging and other I/O operation](/stdlib/io)
- [**Math** — Math helper and structure (Matrix, Vector)](/stdlib/math)
- [**Network** — TCP networking](/stdlib/network)
- [**Std** — Various utilities such as mutex, thread, ...](/stdlib/std)
