# Standard Library

Welcome to the **UnholyC** standard library reference.

The standard library is organized into theses sections:

- [IO](/stdlib/io)  — File, logging and other I/O operation
- [Math](/stdlib/math) — Math helper and structure (Matrix, Vector)
- [Network](/stdlib/network) — TCP networking
- [Std](/stdlib/std) — Various utilities such as mutex, thread, ...
