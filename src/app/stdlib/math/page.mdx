# Math

Math helper and structure (Matrix, Vector)

- [Math](/stdlib/math/math) — Scalar math helper functions
- [Matrix](/stdlib/math/matrix) — 4x4 row-major float matrix
- [VectorF](/stdlib/math/vectorf) — Four-component float vector (x, y, z, w)
- [VectorI](/stdlib/math/vectori) — Four-component integer vector (x, y, z, w)
