<H2 id="create">create</H2>

Create a TCP server bound to a port.

<CodeTabs
  uhc={`TcpServer.It create(I16 port, U16 slots);`}
  cpp={`TcpServer::It create(int16_t port, uint16_t slots);`}
/>

**Parameters**
- `port` — Port to listen on
- `slots` — Maximum number of simultaneous connections

**Returns** — Initialized server object

<H2 id="listen">listen</H2>

Listen for connection and accept them.
This method start a new thread and return new client in a callback

<CodeTabs
  uhc={`U0 listen(TcpServer.It& server, U0 (*callback)(TcpClient.It*));`}
  cpp={`void listen(TcpServer::It& server, void (*callback)(TcpClient::It*));`}
/>

**Parameters**
- `server` — The TCP server
- `callback` — The callback returning new client connection

<H2 id="write">write</H2>

Write a message to a specific client.

<CodeTabs
  uhc={`U8 write(TcpClient.It& client, I8* buffer, U32 size);`}
  cpp={`unsigned char write(TcpClient::It& client, char* buffer, uint32_t size);`}
/>

**Parameters**
- `client` — The target client
- `buffer` — The data to send
- `size` — Number of bytes to send

**Returns** — Non-zero on success

---

<H2 id="read">read</H2>

Read a message from a client.

> [!IMPORTANT]
> This method is non-blocking and may not read all bytes.
> Use `readAll` to guarantee the full payload is received.

<CodeTabs
  uhc={`U8 read(TcpClient.It& client, I8* buffer, U32 size);`}
  cpp={`unsigned char read(TcpClient::It& client, char* buffer, uint32_t size);`}
/>

**Parameters**
- `client` — The source client
- `buffer` — Buffer to receive data into
- `size` — Maximum number of bytes to read

**Returns** — Non-zero on success

---

<H2 id="readAll">readAll</H2>

Read from a client, blocking until all bytes are received.

<CodeTabs
  uhc={`U8 readAll(TcpClient.It& client, I8* buffer, U32 size);`}
  cpp={`unsigned char readAll(TcpClient::It& client, char* buffer, uint32_t size);`}
/>

**Parameters**
- `client` — The source client
- `buffer` — Buffer to receive data into
- `size` — Exact number of bytes to read

**Returns** — Non-zero on success
