<H2 id="create">create</H2>

Create a TCP client and connect it to a server

<CodeTabs
  uhc={`TcpClient.It create(I8* ip, I16 port);`}
  cpp={`TcpClient::It create(char* ip, int16_t port);`}
/>

**Parameters**
- `ip` — Server IP
- `port` — Server port

<H2 id="read">read</H2>

Read a message from the server

> [!IMPORTANT]
> This method is non blocking and all byte may not be read
> To read everything use readAll

<CodeTabs
  uhc={`U8 read(TcpClient.It& client, I8* buffer, U32 size);`}
  cpp={`unsigned char read(TcpClient::It& client, I8* buffer, U32 size);`}
/>

**Parameters**
- `client` — The TCP client
- `buffer` — The data
- `size` — Amount of byte read
