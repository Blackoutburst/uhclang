# File

Easy file reading

---

<H2 id="getSize">getSize</H2>

Returns the size of a file in bytes.

<CodeTabs
  uhc={`I64 getSize(FILE* file);`}
  cpp={`int64_t getSize(FILE* file);`}
/>

**Parameters**
- `file` — the file handle

**Returns** — File size in bytes

---

<H2 id="open">open</H2>

Opens a file and returns a file handle. The file must be explicitly closed after use.

<CodeTabs
  uhc={`FILE* open(const I8* filePath);`}
  cpp={`FILE* open(const char* filePath);`}
/>

**Parameters**
- `filePath` — path to the file

**Returns** — Open file handle, or `null` on failure

---

<H2 id="close">close</H2>

Close and free a file handle.

<CodeTabs
  uhc={`U0 close(FILE* file);`}
  cpp={`void close(FILE* file);`}
/>

**Parameters**
- `file` — the file handle to close

---

<H2 id="read">read</H2>

Read the entire file contents into a heap-allocated string.

<CodeTabs
  uhc={`I8* read(FILE* file);`}
  cpp={`char* read(FILE* file);`}
/>

**Parameters**
- `file` — the file handle

**Returns** — Null-terminated string with the file contents (caller must free)
