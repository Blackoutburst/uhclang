# File

Easy file reading

---

<H2 id="getSize">getSize</H2>

Returns the size of a file in bytes.

<CodeTabs
  uhc={`I64 getSize(FILE* file);`}
  cpp={`int64_t getSize(FILE* file);`}
/>

**Parameters**
- `file` — the file handle

---

<H2 id="open">open</H2>

Opens a file and returns a file handle. The file must be explicitly closed after use.

<CodeTabs
  uhc={`FILE* open(const I8* filePath);`}
  cpp={`FILE* open(const char* filePath);`}
/>

**Parameters**
- `path` — path to the file

---

<H2 id="close">close</H2>

Closes and delete a file handle.

<CodeTabs
  uhc={`U0 close(FILE* file);`}
  cpp={`void close(FILE* file);`}
/>

**Parameters**
- `handle` — the file handle

---

<H2 id="read">read</H2>

Reads the entire contents and return a string of it.

<CodeTabs
  uhc={`I8* read(FILE* file);`}
  cpp={`char* read(FILE* file);`}
/>

**Parameters**
- `handle` — the file handle
