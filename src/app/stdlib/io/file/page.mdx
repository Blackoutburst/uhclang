# File

The `File` namespace provides low-level file system operations: opening, reading, writing, and inspecting files on disk.

---

<h2 id="getSize">getSize</h2>

Returns the size of a file in bytes.

<CodeTabs
  uhc={`I32 File.getSize(U8* path);`}
  cpp={`int getSize(const char* path);`}
/>

**Parameters**
- `path` — absolute or relative path to the file

---

<h2 id="open">open</h2>

Opens a file and returns a file handle. The file must be explicitly closed after use.

<CodeTabs
  uhc={`FileHandle File.open(U8* path, U8* mode);`}
  cpp={`FileHandle* open(const char* path, const char* mode);`}
/>

**Parameters**
- `path` — path to the file
- `mode` — `"r"` (read), `"w"` (write), `"a"` (append)

---

<h2 id="close">close</h2>

Closes an open file handle and flushes any pending writes.

<CodeTabs
  uhc={`U0 File.close(FileHandle handle);`}
  cpp={`void close(FileHandle* handle);`}
/>

**Parameters**
- `handle` — a handle previously returned by `open`

---

<h2 id="read">read</h2>

Reads the entire contents of an open file handle as a string.

<CodeTabs
  uhc={`U8* File.read(FileHandle handle);`}
  cpp={`char* read(FileHandle* handle);`}
/>

**Parameters**
- `handle` — a handle previously returned by `open`
