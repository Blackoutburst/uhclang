# File

Simple file reading utilities

<H2 id="getSize">getSize</H2>

Get the size of an open file in bytes

<CodeTabs
  uhc={`I64 getSize(FILE* file);`}
  cpp={`long long getSize(FILE* file);`}
/>

**Parameters**
- `file` — Open FILE pointer

**Returns** — File size in bytes

---

<H2 id="open">open</H2>

Open a file for reading

<CodeTabs
  uhc={`FILE* open(const I8* filePath);`}
  cpp={`FILE* open(const char* filePath);`}
/>

**Parameters**
- `filePath` — Path to the file

**Returns** — Open FILE pointer, or NULL on failure

---

<H2 id="close">close</H2>

Close an open file

<CodeTabs
  uhc={`U0 close(FILE* file);`}
  cpp={`void close(FILE* file);`}
/>

**Parameters**
- `file` — The FILE pointer to close

---

<H2 id="read">read</H2>

Read the entire file contents into a heap-allocated string

<CodeTabs
  uhc={`I8* read(FILE* file);`}
  cpp={`char* read(FILE* file);`}
/>

**Parameters**
- `file` — Open FILE pointer

**Returns** — Null-terminated file contents — caller must free
