# Thread

Lightweight thread creation and joining

<H2 id="Fn">Fn</H2>

Thread entry point function type

<CodeTabs
  uhc={`typedef U0* (*Fn)(U0*);`}
  cpp={`typedef void* (*Fn)(void*);`}
/>

<H2 id="It">It</H2>

Platform-specific thread handle

<CodeTabs
  uhc={`#if defined(_WIN32) || defined(_WIN64)
  struct It {

#else
  struct It {
    pthread_t handle;
  };
#endif
  `}
  cpp={`#if defined(_WIN32) || defined(_WIN64)
  struct It {
    HANDLE handle;
  };
#else
  struct It {

  `}
/>

---

<H2 id="start">start</H2>

Start a new thread running the given function

<CodeTabs
  uhc={`Thread.It start(Thread.Fn method, U0* arg);`}
  cpp={`Thread::It start(Thread::Fn method, void* arg);`}
/>

**Parameters**
- `method` — The thread entry function
- `arg` — Argument passed to the thread function

**Returns** — Thread handle — use with join

---

<H2 id="join">join</H2>

Wait for a thread to finish

<CodeTabs
  uhc={`U0 join(Thread.It& thread);`}
  cpp={`void join(Thread::It& thread);`}
/>

**Parameters**
- `thread` — The thread to wait on
