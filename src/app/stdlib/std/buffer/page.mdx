# Buffer

Binary buffer with cursor-based reading and writing

<H2 id="It">It</H2>

Buffer object

<CodeTabs
  uhc={`struct It {
    U0* handle
    U8 order
    U32 cursor
};
  `}
  cpp={`struct It {
    void* handle;
    unsigned char order;
    unsigned int cursor;
};
  `}
/>

---

<H2 id="setOrder">setOrder</H2>

Set the byte order for subsequent get/put operations

<CodeTabs
  uhc={`U0 setOrder(Buffer buffer, U8 order)`}
  cpp={`void setOrder(Buffer::It& buffer, unsigned char order);`}
/>

**Parameters**
- `buffer` — The buffer
- `order` — Byte order — use UHC_LITTLE_ENDIAN or UHC_BIG_ENDIAN
