<H2 id="load">load</H2>

Load a dynamic library by name or path.

<CodeTabs
  uhc={`DynamicLibrary.It load(const I8* name);`}
  cpp={`DynamicLibrary::It load(const char* name);`}
/>

**Parameters**
- `name` — The library name or path (e.g. `"libfoo.so"`, `"foo.dll"`)

**Returns** — Library object — check `isValid` before use

---

<H2 id="get">get</H2>

Get a function from a library

<CodeTabs
  uhc={`U0* get(DynamicLibrary.It& dll, const I8* functionName);`}
  cpp={`void* get(DynamicLibrary::It& dll, const char* functionName);`}
/>

**Parameters**
- `dll` — The dynamic library
- `functionName` — The exported function name

**Returns** — Raw function pointer (`void*`) — cast to the expected signature before calling
