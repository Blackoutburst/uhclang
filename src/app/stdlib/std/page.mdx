# STD

Various utilities such as mutex, thread, ...

- [Buffer](/stdlib/std/buffer) — Binary buffer with cursor-based reading and writing
- [DynamicLibrary](/stdlib/std/dynamiclibrary) — Cross-platform shared library loader
- [List](/stdlib/std/list) — Generic doubly-linked list
- [Mutex](/stdlib/std/mutex) — Mutual exclusion primitive for thread synchronization
- [Thread](/stdlib/std/thread) — Lightweight thread creation and joining
