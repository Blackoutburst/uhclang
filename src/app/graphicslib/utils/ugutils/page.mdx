# UgUtils

Miscellaneous Vulkan rendering utilities

<H2 id="setClearColor">setClearColor</H2>

Set the global clear color used at the start of each render pass

<CodeTabs
  uhc={`U0 setClearColor(F32 r, F32 g, F32 b, F32 a);`}
  cpp={`void setClearColor(float r, float g, float b, float a);`}
/>

**Parameters**
- `r` — Red component (0.0–1.0)
- `g` — Green component (0.0–1.0)
- `b` — Blue component (0.0–1.0)
- `a` — Alpha component (0.0–1.0)

---

<H2 id="getClearColor">getClearColor</H2>

Get the currently set global clear color

<CodeTabs
  uhc={`VkClearColorValue getClearColor();`}
  cpp={`VkClearColorValue getClearColor();`}
/>

**Returns** — VkClearColorValue with the stored RGBA values

---

<H2 id="findMemoryType">findMemoryType</H2>

Find a device memory type index satisfying the given requirements

<CodeTabs
  uhc={`U32 findMemoryType(UgContext.It& ctx, U32 typeFilter, VkMemoryPropertyFlags properties);`}
  cpp={`unsigned int findMemoryType(UgContext::It& ctx, unsigned int typeFilter, VkMemoryPropertyFlags properties);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `typeFilter` — Bitmask of acceptable memory type indices
- `properties` — Required memory property flags

**Returns** — Index of a suitable memory type
