# VAO

Vertex array namespaces.

- [UgVertexAttrib](/graphicslib/core/ugvertexattrib) — Single vertex attribute descriptor
- [UgVertexDesc](/graphicslib/core/ugvertexdesc) — Full vertex layout descriptor
- [UgVertexSettings](/graphicslib/core/ugvertexsettings) — Pipeline render settings
- [UgVertexArray](/graphicslib/core/ugvertexarray) — Vertex array object with pipeline
