# Buffer

GPU buffer namespaces.

- [UgBuffer](/graphicslib/core/ugbuffer) — Raw GPU buffer allocation and upload
- [UgFrameBuffer](/graphicslib/core/ugframebuffer) — Framebuffer collection (one per swapchain image)
- [UgUniformBuffer](/graphicslib/core/uguniformbuffer) — Uniform buffer with descriptor set
- [UgCommandBuffer](/graphicslib/core/ugcommandbuffer) — Command buffer recording
