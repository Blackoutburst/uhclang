# UgRenderPass

Vulkan render pass management

<H2 id="end">end</H2>

End the current render pass

<CodeTabs
  uhc={`U0 end(VkCommandBuffer cmdBuffer);`}
  cpp={`void end(VkCommandBuffer cmdBuffer);`}
/>

**Parameters**
- `cmdBuffer` — Active command buffer

---

<H2 id="begin">begin</H2>

Begin a render pass using the framebuffer for the current image index

<CodeTabs
  uhc={`U0 begin(VkRenderPass renderPass, UgFrameBuffer.It& framebuffers, VkCommandBuffer cmdBuffer, U32 imageIndex);`}
  cpp={`void begin(VkRenderPass renderPass, UgFrameBuffer::It& framebuffers, VkCommandBuffer cmdBuffer, unsigned int imageIndex);`}
/>

**Parameters**
- `renderPass` — Render pass to begin
- `framebuffers` — Framebuffer array to source the target from
- `cmdBuffer` — Active command buffer
- `imageIndex` — Index of the current swapchain image

---

<H2 id="destroy">destroy</H2>

Destroy a render pass

<CodeTabs
  uhc={`U0 destroy(UgContext.It& ctx, VkRenderPass renderPass);`}
  cpp={`void destroy(UgContext::It& ctx, VkRenderPass renderPass);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `renderPass` — Render pass to destroy

---

<H2 id="create">create</H2>

Create a render pass compatible with the swapchain's format and depth buffer

<CodeTabs
  uhc={`VkRenderPass create(UgContext.It& ctx, UgSwapchain.It& swapchain);`}
  cpp={`VkRenderPass create(UgContext::It& ctx, UgSwapchain::It& swapchain);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `swapchain` — Swapchain whose format is used for the color attachment

**Returns** — Newly created render pass handle
