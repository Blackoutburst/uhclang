# UgPhysicalDevice

Physical GPU enumeration and selection.

<H2 id="getProperties">getProperties</H2>

Get the properties of a physical device.

<CodeTabs
  uhc={`VkPhysicalDeviceProperties getProperties(VkPhysicalDevice device);`}
  cpp={`VkPhysicalDeviceProperties getProperties(VkPhysicalDevice device);`}
/>

**Parameters**
- `device` — The physical device

**Returns** — Device properties struct

---

<H2 id="typeName">typeName</H2>

Return a human-readable string for a device type.

<CodeTabs
  uhc={`const I8* typeName(VkPhysicalDeviceType type);`}
  cpp={`const char* typeName(VkPhysicalDeviceType type);`}
/>

**Parameters**
- `type` — The Vulkan device type enum value

**Returns** — Null-terminated string (e.g. `"Discrete GPU"`)

---

<H2 id="hasExtensions">hasExtensions</H2>

Check whether a physical device supports the required extensions.

<CodeTabs
  uhc={`U8 hasExtensions(VkPhysicalDevice device);`}
  cpp={`unsigned char hasExtensions(VkPhysicalDevice device);`}
/>

**Parameters**
- `device` — The physical device to check

**Returns** — Non-zero if all required extensions are supported

<H2 id="get">get</H2>

Select a physical device of a desired type.

<CodeTabs
  uhc={`VkPhysicalDevice get(VkInstance instance, VkSurfaceKHR surface, U8 desiredType);`}
  cpp={`VkPhysicalDevice get(VkInstance instance, VkSurfaceKHR surface, unsigned char desiredType);`}
/>

**Parameters**
- `instance` — The Vulkan instance
- `surface` — The window surface (for presentation support check)
- `desiredType` — Preferred `VkPhysicalDeviceType`

**Returns** — Selected physical device
