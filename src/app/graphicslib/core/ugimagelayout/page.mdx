# UgImageLayout

Image layout transition utilities

<H2 id="transition">transition</H2>

Record a pipeline barrier to transition an image between layouts

<CodeTabs
  uhc={`U0 transition(VkCommandBuffer cmdBuffer, VkImage image, VkFormat format, VkImageLayout oldLayout, VkImageLayout newLayout);`}
  cpp={`void transition(VkCommandBuffer cmdBuffer, VkImage image, VkFormat format, VkImageLayout oldLayout, VkImageLayout newLayout);`}
/>

**Parameters**
- `cmdBuffer` — Command buffer to record the barrier into
- `image` — Image whose layout is being transitioned
- `format` — Pixel format of the image
- `oldLayout` — Current image layout
- `newLayout` — Target image layout
