<H2 id="getQueue">getQueue</H2>

Retrieve a queue handle from a logical device.

<CodeTabs
  uhc={`VkQueue getQueue(VkPhysicalDevice device, VkDevice logicalDevice, VkQueueFlagBits queueType);`}
  cpp={`VkQueue getQueue(VkPhysicalDevice device, VkDevice logicalDevice, VkQueueFlagBits queueType);`}
/>

**Parameters**
- `device` — The physical device
- `logicalDevice` — The logical device
- `queueType` — The desired queue capability

**Returns** — Queue handle
