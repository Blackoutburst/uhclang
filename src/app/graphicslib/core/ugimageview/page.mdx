# UgImageView

Vulkan image view creation and management

<H2 id="It">It</H2>

Vulkan image together with its default image view

<CodeTabs
  uhc={`struct It {
    VkImage image;
    VkImageView view;
};
  `}
  cpp={`struct It {
    VkImage image;
    VkImageView view;
};
  `}
/>

---

<H2 id="create">create</H2>

Create an image view for a given image and format

<CodeTabs
  uhc={`UgImageView.It create(UgContext.It& ctx, VkImage image, VkFormat format);`}
  cpp={`UgImageView::It create(UgContext::It& ctx, VkImage image, VkFormat format);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `image` — Image to create the view for
- `format` — Pixel format of the image

**Returns** — Initialized image view object

---

<H2 id="destroy">destroy</H2>

Destroy an image view

<CodeTabs
  uhc={`U0 destroy(UgContext.It& ctx, UgImageView.It& imageView);`}
  cpp={`void destroy(UgContext::It& ctx, UgImageView::It& imageView);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `imageView` — Image view to destroy

---

<H2 id="makeView">makeView</H2>

Create a raw VkImageView with a custom aspect mask

<CodeTabs
  uhc={`VkImageView makeView(UgContext.It& ctx, VkImage image, VkFormat format, VkImageAspectFlags aspectMask);`}
  cpp={`VkImageView makeView(UgContext::It& ctx, VkImage image, VkFormat format, VkImageAspectFlags aspectMask);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `image` — Image to create the view for
- `format` — Pixel format of the image
- `aspectMask` — Aspect flags for the view (color, depth, stencil)

**Returns** — Raw image view handle
