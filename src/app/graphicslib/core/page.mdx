# Core

Vulkan core abstractions for rendering.

- [Buffer](/graphicslib/core/buffer) — GPU buffers, frame buffers, uniform buffers, command buffers
- [CommandPool](/graphicslib/core/commandpool) — Command pool management
- [Context](/graphicslib/core/context) — Vulkan context (instance, device, surface, pools)
- [DescriptorPool](/graphicslib/core/descriptorpool) — Descriptor pool management
- [Device](/graphicslib/core/device) — Physical device, queue families, logical device
- [Fence](/graphicslib/core/fence) — CPU-GPU synchronization fence
- [Image](/graphicslib/core/image) — Image views and layout transitions
- [Instance](/graphicslib/core/instance) — Vulkan instance creation
- [Pipeline](/graphicslib/core/pipeline) — Graphics pipeline and pipeline layout
- [Renderer](/graphicslib/core/renderer) — High-level renderer (swapchain + framebuffers + render pass)
- [RenderPass](/graphicslib/core/renderpass) — Render pass management
- [RenderTarget](/graphicslib/core/rendertarget) — Off-screen render target
- [Semaphore](/graphicslib/core/semaphore) — GPU-GPU synchronization semaphore
- [Shader](/graphicslib/core/shader) — Shader module and shader program
- [Surface](/graphicslib/core/surface) — Window surface
- [Swapchain](/graphicslib/core/swapchain) — Swapchain management
- [Texture](/graphicslib/core/texture) — Textures and samplers
- [VAO](/graphicslib/core/vao) — Vertex attributes, descriptors, settings, and vertex arrays
