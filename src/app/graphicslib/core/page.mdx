# Core

Vulkan core rendering primitives

- [UgContext](/graphicslib/core/context) — Top-level Vulkan context holding all core rendering state
- [UgInstance](/graphicslib/core/instance) — Vulkan instance creation and destruction
- [UgPhysicalDevice](/graphicslib/core/ugphysicaldevice) — Physical device enumeration and selection utilities
- [UgQueueFamilies](/graphicslib/core/ugqueuefamilies) — Queue family query utilities
- [UgDevice](/graphicslib/core/ugdevice) — Logical device creation and queue access
- [UgBuffer](/graphicslib/core/ugbuffer) — Generic GPU buffer allocation and upload
- [UgFrameBuffer](/graphicslib/core/ugframebuffer) — Framebuffer array tied to swapchain images
- [UgUniformBuffer](/graphicslib/core/uguniformbuffer) — Uniform buffer with descriptor set management
- [UgCommandBuffer](/graphicslib/core/ugcommandbuffer) — Single-use and reusable command buffer wrapper
- [UgCommandPool](/graphicslib/core/commandpool) — Vulkan command pool creation and destruction
- [UgDescriptorPool](/graphicslib/core/descriptorpool) — Vulkan descriptor pool creation and destruction
- [UgFence](/graphicslib/core/fence) — CPU–GPU synchronization fence
- [UgSemaphore](/graphicslib/core/semaphore) — GPU–GPU synchronization semaphore
- [UgImageView](/graphicslib/core/ugimageview) — Vulkan image view creation and management
- [UgImageLayout](/graphicslib/core/ugimagelayout) — Image layout transition utilities
- [UgWindowSurface](/graphicslib/core/surface) — Vulkan window surface creation and destruction
- [UgSwapchain](/graphicslib/core/swapchain) — Vulkan swapchain creation and frame management
- [UgShader](/graphicslib/core/ugshader) — SPIR-V shader module loading
- [UgShaderProgram](/graphicslib/core/ugshaderprogram) — Linked vertex and fragment shader program
- [UgRenderPass](/graphicslib/core/renderpass) — Vulkan render pass management
- [UgRenderTarget](/graphicslib/core/rendertarget) — Off-screen render target backed by a texture
- [UgRenderer](/graphicslib/core/renderer) — High-level swapchain renderer managing per-frame synchronization
- [UgPipeline](/graphicslib/core/ugpipeline) — Vulkan graphics pipeline creation helpers
- [UgPipelineLayout](/graphicslib/core/ugpipelinelayout) — Vulkan pipeline layout management
- [UgMaterialPipelineEntry](/graphicslib/core/ugmaterialpipelineentry) — Per-render-pass pipeline cache entry
- [UgMaterial](/graphicslib/core/ugmaterial) — Material combining shaders, vertex layout, and per-pass pipeline cache
- [UgTexture](/graphicslib/core/ugtexture) — 2D texture loading and descriptor management
- [UgSampler](/graphicslib/core/ugsampler) — Vulkan sampler creation and destruction
- [UgTextureArray](/graphicslib/core/ugtexturearray) — Layered texture array for batch rendering
- [UgVertexAttrib](/graphicslib/core/ugvertexattrib) — Single vertex attribute binding description
- [UgVertexDesc](/graphicslib/core/ugvertexdesc) — Vertex buffer layout description
- [UgVertexSettings](/graphicslib/core/ugvertexsettings) — Rasterizer, blending, and depth settings for a pipeline
