<H2 id="create">create</H2>

Create a vertex array object with a fully built pipeline.

<CodeTabs
  uhc={`UgVertexArray.It create(UgContext.It& ctx, UgSwapchain.It& swapchain, VkRenderPass renderPass, UgVertexDesc.It& desc, UgUniformBuffer.It& uniformBuffer, UgTexture.It* texture, UgShaderProgram.It& shaderProgram, const UgVertexSettings.It& settings);`}
  cpp={`UgVertexArray::It create(UgContext::It& ctx, UgSwapchain::It& swapchain, VkRenderPass renderPass, UgVertexDesc::It& desc, UgUniformBuffer::It& uniformBuffer, UgTexture::It* texture, UgShaderProgram::It& shaderProgram, const UgVertexSettings::It& settings);`}
/>

**Parameters**
- `ctx` — The Vulkan context
- `swapchain` — The swapchain (for viewport size)
- `renderPass` — The compatible render pass
- `desc` — Vertex layout descriptor
- `uniformBuffer` — The uniform buffer for this VAO
- `texture` — Optional texture (pass `null` for untextured geometry)
- `shaderProgram` — Vertex and fragment shaders
- `settings` — Pipeline render settings

**Returns** — Initialized vertex array object
