# RenderTarget

Off-screen render target backed by a texture.

<H2 id="It">It</H2>

Render target object.

<CodeTabs
  uhc={`struct It {
    VkRenderPass renderPass;
    VkFramebuffer framebuffer;
    UgTexture.It texture;
    U32 width;
    U32 height;
  };`}
  cpp={`struct It {
    VkRenderPass renderPass;
    VkFramebuffer framebuffer;
    UgTexture::It texture;
    uint32_t width;
    uint32_t height;
  };`}
/>

---

<H2 id="create">create</H2>

Create an off-screen render target of the given dimensions.

<CodeTabs
  uhc={`UgRenderTarget.It create(UgContext.It& ctx, U32 width, U32 height);`}
  cpp={`UgRenderTarget::It create(UgContext::It& ctx, uint32_t width, uint32_t height);`}
/>

**Parameters**
- `ctx` — The Vulkan context
- `width` — Width in pixels
- `height` — Height in pixels

**Returns** — Initialized render target
