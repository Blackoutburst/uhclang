# UgRenderTarget

<CodeTabs
  uhc={`struct It {
    VkRenderPass renderPass;
    VkFramebuffer framebuffer;

    U32 height;
};
  `}
  cpp={`struct It {
    VkRenderPass renderPass;

    UgTexture::It texture;

    unsigned int height;
};
  `}

<H2 id="create">create</H2>

Create an off-screen render target of the given dimensions

<CodeTabs
  uhc={`UgRenderTarget.It create(UgContext.It& ctx, U32 width, U32 height);`}
  cpp={`UgRenderTarget::It create(UgContext::It& ctx, unsigned int width, unsigned int height);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `width` — Width in pixels
- `height` — Height in pixels

**Returns** — Initialized render target object
