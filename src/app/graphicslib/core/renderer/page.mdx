# Renderer

High-level renderer bundling the swapchain, framebuffers, and render pass.

<H2 id="It">It</H2>

Renderer object.

<CodeTabs
  uhc={`struct It {
    UgSwapchain.It swapchain;
    UgFrameBuffer.It framebuffers;
    VkRenderPass renderPass;
  };`}
  cpp={`struct It {
    UgSwapchain::It swapchain;
    UgFrameBuffer::It framebuffers;
    VkRenderPass renderPass;
  };`}
/>

<H2 id="create">create</H2>

Create a renderer with a swapchain, framebuffers, and render pass.

<CodeTabs

/>
