# UgBuffer

Generic GPU buffer allocation and upload

<H2 id="It">It</H2>

GPU buffer with its backing device memory

<CodeTabs
  uhc={`struct It {
    VkBuffer handle;
    VkDeviceMemory memory;
};
  `}
  cpp={`struct It {
    VkBuffer handle;
    VkDeviceMemory memory;
};
  `}
/>

---

<H2 id="create">create</H2>

Allocate a GPU buffer with the specified usage

<CodeTabs
  uhc={`UgBuffer.It create(UgContext.It& ctx, U32 size, VkBufferUsageFlagBits usage);`}
  cpp={`UgBuffer::It create(UgContext::It& ctx, unsigned int size, VkBufferUsageFlagBits usage);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `size` — Size in bytes
- `usage` — Buffer usage flags

**Returns** — Allocated buffer object

---

<H2 id="upload">upload</H2>

Upload data from CPU memory into a GPU buffer

<CodeTabs
  uhc={`U0 upload(UgContext.It& ctx, UgBuffer.It& buffer, U0* data, U32 size);`}
  cpp={`void upload(UgContext::It& ctx, UgBuffer::It& buffer, void* data, unsigned int size);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `buffer` — Destination buffer
- `data` — Pointer to source data
- `size` — Number of bytes to upload

---

<H2 id="destroy">destroy</H2>

Destroy a GPU buffer and free its device memory

<CodeTabs
  uhc={`U0 destroy(UgContext.It& ctx, UgBuffer.It& buffer);`}
  cpp={`void destroy(UgContext::It& ctx, UgBuffer::It& buffer);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `buffer` — Buffer to destroy
