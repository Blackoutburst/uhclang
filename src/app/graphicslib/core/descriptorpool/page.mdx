# UgDescriptorPool

Vulkan descriptor pool creation and destruction

<H2 id="create">create</H2>

Create a descriptor pool sized for uniform buffers and combined image samplers

<CodeTabs
  uhc={`VkDescriptorPool create(UgContext.It& ctx);`}
  cpp={`VkDescriptorPool create(UgContext::It& ctx);`}
/>

**Parameters**
- `ctx` — Vulkan context

**Returns** — Newly created descriptor pool handle

---

<H2 id="destroy">destroy</H2>

Destroy a descriptor pool and free all sets allocated from it

<CodeTabs
  uhc={`U0 destroy(UgContext.It& ctx, VkDescriptorPool descriptorPool);`}
  cpp={`void destroy(UgContext::It& ctx, VkDescriptorPool descriptorPool);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `descriptorPool` — Descriptor pool to destroy
