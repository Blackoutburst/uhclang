# UgInstance

Vulkan instance creation and destruction

<H2 id="create">create</H2>

Create a Vulkan instance

<CodeTabs
  uhc={`VkInstance create(U8 debug);`}
  cpp={`VkInstance create(unsigned char debug);`}
/>

**Parameters**
- `debug` — Non-zero to enable validation layer support

**Returns** — Newly created VkInstance handle

---

<H2 id="destroy">destroy</H2>

Destroy the Vulkan instance stored in the context

<CodeTabs
  uhc={`U0 destroy(UgContext.It& ctx);`}
  cpp={`void destroy(UgContext::It& ctx);`}
/>

**Parameters**
- `ctx` — Vulkan context whose instance will be destroyed
