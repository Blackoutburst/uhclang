# UgPipelineLayout

Pipeline layout creation from descriptor set layouts.

<H2 id="create">create</H2>

Create a pipeline layout from an array of descriptor set layouts.

<CodeTabs
  uhc={`VkPipelineLayout create(UgContext.It& ctx, U32 count, VkDescriptorSetLayout* setLayouts);`}
  cpp={`VkPipelineLayout create(UgContext::It& ctx, uint32_t count, VkDescriptorSetLayout* setLayouts);`}
/>

**Parameters**
- `ctx` — The Vulkan context
- `count` — Number of descriptor set layouts
- `setLayouts` — Array of descriptor set layouts

**Returns** — Created `VkPipelineLayout`
