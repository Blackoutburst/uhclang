# UgVertexSettings

Pipeline render settings controlling blending, culling, depth, and polygon mode. All fields have sensible defaults.

**Key fields**
- `depthVao` — Non-zero to enable depth-only rendering
- `polygonMode` — Fill, line, or point rendering
- `cullMode` — Face culling mode
- `frontFace` — Winding order for front faces
- `blendEnable` — Toggle alpha blending
- `lineWidth` — Line width when `polygonMode` is `VK_POLYGON_MODE_LINE`
