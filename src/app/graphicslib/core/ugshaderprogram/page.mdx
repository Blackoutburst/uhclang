# UgShaderProgram

Linked vertex and fragment shader program

<H2 id="It">It</H2>

Vertex and fragment shader pair with their pipeline stage info

<CodeTabs
  uhc={`struct It {
    VkPipelineShaderStageCreateInfo handle[2];
    VkShaderModule vertexShader;
    VkShaderModule fragmentShader;
};
  `}
  cpp={`struct It {
    VkPipelineShaderStageCreateInfo handle[2];
    VkShaderModule vertexShader;
    VkShaderModule fragmentShader;
};
  `}
/>

---

<H2 id="createShaderInfo">createShaderInfo</H2>

Build the pipeline shader stage create info for a single stage

<CodeTabs
  uhc={`VkPipelineShaderStageCreateInfo createShaderInfo(VkShaderStageFlagBits stage, VkShaderModule shader);`}
  cpp={`VkPipelineShaderStageCreateInfo createShaderInfo(VkShaderStageFlagBits stage, VkShaderModule shader);`}
/>

**Parameters**
- `stage` — Shader stage flag (vertex or fragment)
- `shader` — Shader module to attach

**Returns** — Populated pipeline shader stage create info

---

<H2 id="create">create</H2>

Create a shader program from vertex and fragment SPIR-V files

<CodeTabs
  uhc={`UgShaderProgram.It create(UgContext.It& ctx, const I8* vertexSource, const I8* fragmentSource);`}
  cpp={`UgShaderProgram::It create(UgContext::It& ctx, const char* vertexSource, const char* fragmentSource);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `vertexSource` — Path to the compiled vertex SPIR-V binary
- `fragmentSource` — Path to the compiled fragment SPIR-V binary

**Returns** — Initialized shader program object
