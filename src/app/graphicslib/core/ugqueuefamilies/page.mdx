# UgQueueFamilies

Query queue families on a physical device.

<H2 id="count">count</H2>

Return the number of queue families on a physical device.

<CodeTabs
  uhc={`U32 count(VkPhysicalDevice device);`}
  cpp={`uint32_t count(VkPhysicalDevice device);`}
/>

**Parameters**
- `device` — The physical device

**Returns** — Number of queue families

---

<H2 id="hasType">hasType</H2>

Check if a physical device has a queue family of the given type.

<CodeTabs
  uhc={`U8 hasType(VkPhysicalDevice device, VkQueueFlagBits type);`}
  cpp={`unsigned char hasType(VkPhysicalDevice device, VkQueueFlagBits type);`}
/>

**Parameters**
- `device` — The physical device
- `type` — The queue capability flags to look for

**Returns** — Non-zero if a matching family exists

---

<H2 id="getType">getType</H2>

Get the index of the first queue family matching a type.

<CodeTabs
  uhc={`U32 getType(VkPhysicalDevice device, VkQueueFlagBits type);`}
  cpp={`uint32_t getType(VkPhysicalDevice device, VkQueueFlagBits type);`}
/>

**Parameters**
- `device` — The physical device
- `type` — The queue capability flags

**Returns** — Queue family index
