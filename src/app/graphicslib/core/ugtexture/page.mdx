<H2 id="create">create</H2>

<CodeTabs
  uhc={`UgTexture.It create(UgContext.It& ctx, const I8* path);`}
  cpp={`UgTexture::It create(UgContext::It& ctx, const char* path);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `path` — Path to the image file

<H2 id="bind">bind</H2>

Bind the texture's descriptor set to a pipeline slot

<CodeTabs
  uhc={`U0 bind(UgTexture.It& texture, UgCommandBuffer.It& cmd, VkPipelineLayout pipelineLayout, U32 slot);`}
  cpp={`void bind(UgTexture::It& texture, UgCommandBuffer::It& cmd, VkPipelineLayout pipelineLayout, unsigned int slot);`}
/>

**Parameters**
- `texture` — Texture to bind
- `cmd` — Active command buffer
- `pipelineLayout` — Pipeline layout the descriptor set is compatible with
- `slot` — Descriptor set index (texture slot)
