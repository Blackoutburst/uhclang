# UgUniformBuffer

Uniform buffer with descriptor set management

<H2 id="It">It</H2>

Uniform buffer with its descriptor set and mapped CPU pointer

<CodeTabs
  uhc={`struct It {
    UgBuffer.It buffer;
    VkDescriptorSetLayout setLayout;
    U0* mappedData;
    U32 binding;
    U32 size;
    VkDescriptorSet descriptorSet;
    U32 cursor;
};
  `}
  cpp={`struct It {
    UgBuffer::It buffer;
    VkDescriptorSetLayout setLayout;
    void* mappedData;
    unsigned int binding;
    unsigned int size;
    VkDescriptorSet descriptorSet;
    unsigned int cursor;
};
  `}
/>

<H2 id="create">create</H2>

Create a uniform buffer and allocate its descriptor set

<CodeTabs
  uhc={`UgUniformBuffer.It create(UgContext.It& ctx, U32 binding, U32 size);`}
  cpp={`UgUniformBuffer::It create(UgContext::It& ctx, unsigned int binding, unsigned int size);`}
/>

**Parameters**
- `ctx` — Vulkan context
- `binding` — Descriptor binding index
- `size` — Size of the uniform buffer in bytes

**Returns** — Initialized uniform buffer

<H2 id="bind">bind</H2>

Bind the uniform buffer's descriptor set to the command buffer

<CodeTabs
  uhc={`U0 bind(UgUniformBuffer.It& uniformBuffer, VkCommandBuffer cmdBuffer, VkPipelineLayout pipelineLayout);`}
  cpp={`void bind(UgUniformBuffer::It& uniformBuffer, VkCommandBuffer cmdBuffer, VkPipelineLayout pipelineLayout);`}
/>

**Parameters**
- `uniformBuffer` — Uniform buffer to bind
- `cmdBuffer` — Active command buffer
- `pipelineLayout` — Pipeline layout the descriptor set is compatible with
