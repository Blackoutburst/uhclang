# UgUniformBuffer

Uniform buffer backed by a descriptor set for shader data upload.

<H2 id="It">It</H2>

Uniform buffer object.

<CodeTabs
  uhc={`struct It {
    UgBuffer.It buffer;
    VkDescriptorSetLayout setLayout;
    U0* mappedData;
    U32 binding;
    U32 size;
    VkDescriptorSet descriptorSet;
    U32 cursor;
  };`}
  cpp={`struct It {
    UgBuffer::It buffer;
    VkDescriptorSetLayout setLayout;
    void* mappedData;
    uint32_t binding;
    uint32_t size;
    VkDescriptorSet descriptorSet;
    uint32_t cursor;
  };`}
/>

<H2 id="create">create</H2>

Create a uniform buffer at a given shader binding point.

<CodeTabs
  uhc={`UgUniformBuffer.It create(UgContext.It& ctx, U32 binding, U32 size);`}
  cpp={`UgUniformBuffer::It create(UgContext::It& ctx, uint32_t binding, uint32_t size);`}
/>

**Parameters**
- `ctx` — The Vulkan context
- `binding` — Shader binding index
- `size` — Buffer size in bytes

**Returns** — Initialized uniform buffer
